# Eval Framework — Methodology & Results

How we measure LLM output quality for health-sensitive food recommendations.

## Eval Suites
- **Health Safety**: Ensures condition-aware recommendations are safe
- **Hallucination Detection**: Catches invented nutrient data
- **Cultural Accuracy**: Validates cuisine-matched suggestions
- **Tone**: Enforces additive, non-judgmental language
- **Parsing Accuracy**: Measures NLP parsing quality
